## Shallow copy vs Deep copy
* Припустимо є таке наслідування: A1 -> C1
* A2 це - копія A1
* `Shallow copy`
  * A1 -> C1 <- A2
* `Deep copy`
  * A1 -> C1
  * A2 -> C2
